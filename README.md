# HDServerConfig

Утилита командной строки для настройки LED-контроллеров Huidu (серия HD-A3L) через SDK 3.0.

## Технические детали

Программа реализует **ручной цикл авторизации** в протоколе Huidu SDK 3.0 (v0x1000007), обходя ограничения стандартной DLL:

1.  **Binary Handshake (0x2001)**: 8-байтовый пакет запроса версии. 
    *   Запрос: `08 00 01 20 07 00 00 01` (Версия 0x1000007).
    *   Ответ: `08 00 02 20 06 00 00 01` (Контроллер подтверждает v0x1000006).
2.  **XML Packet Header (0x2003)**: 12-байтовый заголовок для всех XML команд.
    *   `Len` (2 байта): 12 + XML Length.
    *   `Cmd` (2 байта): 0x2003.
    *   `TotalSize` (4 байта): Общая длина XML контента.
    *   `Index` (4 байта): Смещение (всегда 0 для малых пакетов).
3.  **GUID Auth**: Получение сессионного ключа через метод `GetIFVersion`. В текущей версии реализована автоматическая очистка дефисов из GUID перед отправкой команд.

## Использование

### Проверка настроек и ID устройства
```powershell
.\HDServerConfig.exe verify 192.168.8.2
```

### Установка адреса сервера
```powershell
.\HDServerConfig.exe set 192.168.8.2 77.42.66.37 10001
```

### Удаленная перезагрузка
```powershell
.\HDServerConfig.exe reboot 192.168.8.2
```

## Известные проблемы
*   **kUnsupportMethod**: Ошибка может возникать, если GUID передан в неверном формате (с дефисами) или если команда не поддерживается данной аппаратной ревизией контроллера.
